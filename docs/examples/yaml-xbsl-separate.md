# Пример YMAL/XBSL в разных окнах

Ниже пример двух отдельных блоков кода `yaml` и `xbsl`, которые должны рендериться через наш кастомный highlighter.

```yaml
ВидЭлемента: Справочник
ОбластьВидимости: Глобально
Ид: 8e6ca4cf-fef8-46cf-a0d2-68abd1577dbf
Имя: Изображения
Реквизиты:
    -
        Имя: Наименование
    -
        Имя: Владелец
        Тип: ВладелецИзображения.Ссылка?
    -
        Ид: 00e88485-92fe-4547-b73a-b6514d551084
        Имя: Ориентация
        Тип: ОриентацияИзображения?
    -
        Ид: 271832dc-5149-4e26-b74b-d36cd3260e0e
        Имя: ДвоичныйОбъект
        Тип: ДвоичныйОбъект.Ссылка?
    -
        Ид: 0109bee0-2385-4d5f-a480-3eb0815b7628
        Имя: Миниатюра64
        Тип: ДвоичныйОбъект.Ссылка?
    -
        Ид: 0c17184a-b273-44e7-bb2f-dca3a0bd1ba5
        Имя: Миниатюра128
        Тип: ДвоичныйОбъект.Ссылка?
    -
        Ид: 54737661-6adc-4f5e-be62-f409a0dd9e1e
        Имя: ВысотаИзображения
        Тип: Число
    -
        Ид: 28605480-403f-426f-bbd0-19deb7eb2212
        Имя: ШиринаИзображения
        Тип: Число
```

```xbsl
@Глобально @НаСервере
метод ПолучитьВсе(Владелец: ВладелецИзображения.Ссылка): Массив<Изображения.Ссылка>
    
    знч Запрос = Запрос{
        ВЫБРАТЬ
            Изображения.Ссылка КАК Ссылка
        ИЗ
            Изображения КАК Изображения
        ГДЕ
            Изображения.Владелец == %Владелец
    }
    
    возврат Запрос.Выполнить().ВМассив().Преобразовать(СтрокаДанныхИзображения -> СтрокаДанныхИзображения.Ссылка)
    
;
```
