# Как создать двоичный объект из Base64

**Дано:**

Получена строка в формате base64, хранящая бинарные данные файла. Необходимо сохранить эти данные в базу и получить файл двоичного объекта.

**Решение:**

Для загрузки файла из строки в формате base64 используется следующий код:

```xbsl
знч ДанныеКартинкиСтрокой: Строка // Некоторая строка в формате base64
знч ДвоичныеДанныеОбъект: ДвоичныйОбъект = ОбъектноеХранилище.ЗагрузитьИзBase64Строки(ДанныеКартинкиСтрокой)
```

Более подробно об ОбъектноеХранилище - в [документации](https://1cmycloud.com/console/help/element/7.0/docs/stdlib/element/Std/ObjectStorage/ObjectStorage_ru/)
