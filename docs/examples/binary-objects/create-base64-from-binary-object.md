# Как получить Base64 из двоичного объекта

**Дано:**

Есть ссылка на двоичный объект. Необходимо из него получить строку в формате base64

**Решение:**

Для получения строки в формате base64 из двоичного объекта используется следующий код:

```xbsl
знч СсылкаДО: ДвоичныйОбъект.Ссылка // Ссылка на двоичный объект, полученный извне
Кодировки.Base64.КодироватьВСтроку(СсылкаДО.ЗагрузитьОбъект().ПолучитьБайты())
```

**Предупреждение!** Если файл больше 1 мб, то будет вызвана ошибка чтения данных в память. Для работы с файлами больше 1 мб рекомендуется использовать ПотокЧтения и считывать порциями, кратными 3 байтам для избежания операций выравнивания между частями строки Base64.
