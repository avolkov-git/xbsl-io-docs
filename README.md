# xbsl-docs

Репозиторий с исходниками документации для `xbsl.io`.

## Структура

- `docs/` — документация, которую сайт будет зеркалить в локальный `content/`
- `README.md` — страница репозитория на GitHub, не используется как страница сайта

## Разделы

- `libraries` — документация по библиотекам
- `vs-code-extension` — документация по расширению VS Code
- `examples` — прикладные примеры

## Правила

- каждый `.md` файл внутри раздела — отдельная страница
- `index.md` внутри раздела — главная страница раздела
- порядок top-level разделов задается в `docs/_meta.ts`

1. `libraries`
2. `vs-code-extension`
3. `examples`
